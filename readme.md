# 設定

1. 點開 『選項』
![image](https://media.discordapp.net/attachments/921704490778894368/922877004578172948/unknown.png)

2. 填入最新的金鑰來更新系統
![image](https://media.discordapp.net/attachments/921704490778894368/923938294662856734/unknown.png)
